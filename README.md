# IOTecnologye - Showcase de Automação Residencial High-End

Um showcase imersivo para um projeto de automação residencial de alto padrão, focado em tecnologia, design e experiência do usuário.

## 🚀 Tecnologias Utilizadas

- **React 19**
- **Vite**
- **Tailwind CSS**
- **Framer Motion** (Animações complexas e cinematográficas)
- **Lucide React** (Ícones)
- **Vercel Speed Insights**

## ✨ Funcionalidades

- **Design Premium**: Estética Dark Mode com glassmorphism e efeitos de neon.
- **Scroll Reveal**: Textos e elementos que se revelam conforme a rolagem.
- **Arquitetura vs Engenharia**: Slider interativo para visualização de infraestrutura técnica.
- **Catálogo Interativo**: Galeria de hardware com animações 3D e detalhes técnicos.
- **Mapeamento Espacial**: Planta baixa interativa com hotspots informativos.

## 🛠️ Como executar localmente

1. Clone o repositório
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
